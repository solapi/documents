# Overview

Message V4 에서는 [API Key 인증방식](api-key.md) 과 [OAuth2 인증방식](oauth2.md) 두가지 인증방식을 제공합니다.

HTTP 헤더에 Authorization 을 추가하여 인증정보를 서버에 전달 할 수 있습니다.

아래는 API Key 인증방식과 OAuth2 인증방식의 예입니다.
