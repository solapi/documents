# 메시지상태코드
숫자 4자리로 구성된 메시지의 상태를 나타내는 코드입니다. 코드 첫번째 자리 숫자의 의미는 아래와 같습니다.

| 코드 | 내용 |
| ------------- |-------------|
| 1xxx | 접수(전) 상태 코드) |
| 2xxx | 쿨에스엠에스 게이트웨이 내에서 발송중 상태 코드 |
| 3xxx | 연동망에서 발송 상태 코드 |
| 4xxx | 발송완료 상태 코드 |
