### Group Status

* `PENDING` - 대기중 \(그룹 전송 준비\)
* `SENDING` - 전송 요청됨
* `COMPLETE` - 처리 완료
* `FAILED` - 생성 실패
* `DELETED` - 삭제됨
* `SCHEDULED` - 발송 예약 됨 \(예약 취소전까지 그룹 삭제 불가\)
